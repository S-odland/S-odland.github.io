Swarm and Multi-Robot Systems

Swarm Robotics Class at NU

This was a swarm robotics class in which we read various research papers on swarm algorithms, systems and implementations. We learned about localization algorithms, genetic algorithms, flight algorithms, flocking algorithms and many more. We also implemented a few of these algorithms in the Docker environment. Specifically, we designed a hop count algorithm, an algorithm to mimic the brazil nut effect, and and a flocking algorithm.
See the YouTube link below for the flocking simulation and see the GitHub link for code and more simulations

Language: Python
Environment: Docker

Robotic Manipulation

Introductory Robotic Manipulation Course at NU

This was an introductory course in Robotic Manipulation culminating in a final capstone project where I was tasked with having a YouBot (UR5 arm attached to an omnidirectional base) solve a simple pick and place problem. In this course I learned Feedforward and PID control for robotic arms and bases, using various linear algebraic properties to solve these tasks as the number of components scaled up.
If you'd like to see the results of this capstone or the code that made it work, follow the GitHube and YouTube links below.

Language: Python
Environment: CoppeliaSim
Skills: PID Control, Linear Algebra

Swarm Robotics Research

Interactive and Emergent Autonomy Lab

A swarm robotics experiment intended to mimic a bayesian particle filter experiment using a centralized controller. Continuing from the work of a former grad student, I've worked to enhance the target detect algorithm in the robots and to reduce traffic buildup in the maps during experimentation.
Current experiments function with nine robots. In the future we are looking to expand this to 30 robots so stay tuned. Follow the GitHub link below if you'd look to learn more!

Language: C, Python
Environment: Code Composer Studio

NURC Workshop

Northwestern University Robotics Club

This was an introductory robotics course I designed for new members of NURC. It covers the basics of embedded programming in C++ in Arduino IDE. In this course we covered PWM, simple circuit design, and embedded programming. The culmination of this workshop was in making a motion sensor use the knowledge learned throughout the workshop. The motion sensor was made up of an ESP32, phototransistor, retroreflective tape and a laser.
If you'd like to see the contents of the workshop see the wiki section of the GitHub link below.

Language: C++
Environment: Arduino IDE
Skills: Circuit Design, Embedding Programming

BattleBots

NURC Project for Robobrawl Competition

An annual competition held at UIC, we built a 30 lb. drumbot to compete against other 30 lb. battlebots. The robot was designed in OnShape (an online CAD platform) and made out of aluminum and steel parts. Over 30 hours was put into machinging the robot and around 5-10 hours was spend programming the differential drive. We had to map the movements of a joystick to the drive of the robot.
Follow the youtube links below for a video of the battlebot moving and the weapon spinning up!

Language: C++
Environment: Arduino IDE
Skills: Manual/Conversational Mill, Lathe, Waterjet, Embedded Programming, CAD (OnShape)

Mechatronics

Mechatronics Courses at NU

This spans two mechatronics course at NU, in which we use a PIC32 microcontroller to learn how to communicate between sensors (I2C, SPI), wire up those sensors at make valuable uses of them. For the introductory course, the culmination was in the form of a PID motor control project in which was had a motor go to various angles along its trajectory. In the Advanced course, was just experimented with various electrical components such as an OLED display and accelerometer.
