Swarm Robotics Research

Interactive and Emergent Autonomy Lab

A swarm robotics experiment intended to mimic a bayesian particle filter experiment using a centralized controller. Continuing from the work of a former grad student, I've worked to enhance the target detect algorithm in the robots and to reduce traffic buildup in the maps during experimentation.
Current experiments function with nine robots. In the future we are looking to expand this to 30 robots so stay tuned. Follow the GitHub link below if you'd look to see more!

Language: C, Python
Environment: Code Composer Studio

Robotic Manipulation

Introductory Robotic Manipulation Course at NU

This was an introductory course in Robotic Manipulation culminating in a final capstone project where I was tasked with having a YouBot (UR5 arm attached to an omnidirectional base) solve a simple pick and place problem. In this course I learned Feedforward and PID control for robotic arms and bases, using various linear algebraic properties to solve these tasks as the number of components scaled up.
If you'd like to see the results of this capstone or the code that made it work, follow the GitHube and YouTube links below.
Language: Python
Environment: CoppeliaSim
